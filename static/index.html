What is x402?

x402 is a protocol for stateless, per-request payments over HTTP. Clients attach a signed payment payload to each request. The facilitator verifies it, executes the on-chain transfer, and grants access to protected resourcesâ€”all without accounts, wallets, or user registration.

API Endpoints

GET /health Check facilitator health status
GET /supported List supported payment schemes and networks
POST /verify Verify payment signature and requirements
POST /settle Execute payment on-chain via EIP-3009
